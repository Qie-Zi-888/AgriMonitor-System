# 基于OpenHarmony的智慧农业大棚监控系统
BearPi-HM_Nano 智慧农业案例

## 📌 开发板简介

**BearPi-HM_Nano** 是一款专为 **HarmonyOS** 设计的开发板，主要特性如下：

| 特性 | 说明 |
|------|------|
| **主控芯片** | Hi3861（2.4GHz WLAN SoC） |
| **板载功能** | NFC电路、标准E53接口 |
| **扩展能力** | 支持智慧农业、智能台灯、智能安防、智能烟感等案例 |

E53接口标准使得开发板可以快速连接各种传感器扩展板，实现不同场景的应用。

---

## 🌱 介绍

如何在 **BearPi-HM_Nano** 开发板上：

- 使用 **MQTT协议** 连接 **华为IoT平台**
- 搭配 **E53_IA1 智慧农业扩展板**
- 实现完整的 **智慧农业** 数据采集与上报案例

## 📦 硬件组成

| 组件 | 说明 |
|------|------|
| BearPi-HM_Nano 开发板 | 主控，运行HarmonyOS |
| E53_IA1 扩展板 | 智慧农业专用，集成温湿度、土壤湿度等传感器 |
| 传感器（板载） | 温湿度传感器、光照传感器、土壤湿度传感器 |

---

## ✨ 核心功能

- ✅ 实时采集土壤湿度、空气温湿度、光照强度
- ✅ 通过MQTT协议上报数据到华为IoT平台
- ✅ 平台侧可下发命令（如控制灌溉）
- ✅ 支持远程监控农业环境
